# 最終結果ファイル

このディレクトリには、最適化の最終結果ファイルが保存されます。

## ファイル形式

- `graduated_reward_optimization_YYYYMM_YYYYMMDD_HHMMSS_final.json`
  - 最適化の最終結果
  - 最良パラメータ、全試行結果、統計情報

## 自動コミット

このディレクトリ内のファイルは自動的にGitのコミット対象となります。
重要な結果ファイルとして管理されます。

## ファイル内容

各最終結果ファイルには以下の情報が含まれます：

- **最適化情報**: データ月、試行回数、テストモード
- **最良試行**: 最良の試行番号、スコア、パラメータ
- **全試行結果**: 全ての試行の詳細情報
- **統計情報**: 実行時間、データペア数など

## 使用方法

```bash
# 結果ファイルの確認
ls final_results/

# 最新の結果ファイルを確認
ls -t final_results/ | head -1
```

## 重要な結果ファイル

このディレクトリのファイルは以下の理由で重要です：

1. **最適化の完全な記録**: 全ての試行結果を保持
2. **再現性の確保**: 同じ条件での再実行が可能
3. **比較分析**: 異なるデータ月での結果比較
4. **進捗追跡**: 最適化の進捗と改善状況
