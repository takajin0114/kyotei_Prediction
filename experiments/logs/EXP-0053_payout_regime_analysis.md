# EXP-0053: Payout / Odds Regime Analysis

## 目的

EXP-0052 で「負けブロックは hit_rate 高め・bet 数少なめ・EV 同程度なのに利益が悪化」と確認された。本実験では、利益の強弱が**的中時オッズ分布・払戻分布・hit 1件あたり利益**の差で説明できるかを検証する。

## 条件

- **selection**: d_hi475（skip_top20pct, 4.30≤EV<4.75, prob≥0.05）
- **運用**: switch_dd4000 主対象、比較用に normal_only
- **対象期間**: n_windows=36
- **block_size**: 6 windows（EXP-0052 と整合）

## DB 確認

- 使用 DB: `kyotei_predictor/data/kyotei_races.sqlite`（存在確認済み）
- 予測: `outputs/ev_cap_experiments/rolling_roi_predictions` の既存結果を利用

## 実行コマンド

```bash
python3 -m kyotei_predictor.tools.run_exp0053_payout_regime_analysis --n-windows 36
```

## 勝ち block vs 負け block 比較（switch_dd4000）

| 指標               | 勝ち block (2件) | 負け block (4件) |
|--------------------|------------------|------------------|
| avg_hit_odds       | 18.44            | 8.52             |
| median_hit_odds    | 7.3              | 5.88             |
| avg_payout_per_hit | 1,810.06         | 734.48           |
| avg_profit_per_hit | 1,715.72         | 649.27           |
| avg_bet_count      | 189              | 163.2            |
| avg_hit_rate       | 7.36%            | 8.34%            |
| avg_ev             | 4.5316           | 4.5268           |

## 解釈

1. **弱い局面では的中時オッズが低い**: 勝ち block の avg_hit_odds 18.44 に対し、負け block は 8.52。median も 7.3 vs 5.88 で負け block が低い。
2. **利益悪化は高配当ヒット不足で説明できる**: 負け block は payout_per_hit_mean が約半分（734 vs 1,810）、profit_per_hit も約 1/2.6（649 vs 1,716）。hit_rate は負け block の方が高い（8.34% vs 7.36%）ため、「当たらない」のではなく「当たっても低配当に寄っている」ことが数値で確認された。
3. **EV は block 間でほぼ同水準**（4.53 前後）であり、選別条件の効き方ではなく、**払戻レジーム（的中時のオッズ分布）の差**が利益差の主因と解釈できる。

## 採用判断

- **judgment**: **hold**（分析実験として知見を採用、運用変更は行わない）
- **理由**: 「弱い局面では的中時オッズが低い」「利益悪化は高配当ヒット不足で説明できる」という仮説を**明確に支持**。知見は chat_context / project_status に反映するが、本 EXP 単体ではオッズフィルタ等の即時運用変更は行わない。
