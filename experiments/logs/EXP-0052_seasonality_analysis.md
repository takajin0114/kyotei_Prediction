# Experiment: EXP-0052 Seasonality / Regime Analysis

## experiment id

EXP-0052

## purpose

EXP-0051 で後半（7月後半〜10月）の大幅赤字が確認されたため、「季節性」または「市場レジーム」を明確にする。月別分析・block 分析・累積利益曲線・switch 発動・regime 特徴（bet_count / hit_rate / EV 帯）を出力し、どの期間が強く・どの期間が弱いかを整理する。

## DB 存在確認結果

- **確認パス**: `kyotei_predictor/data/kyotei_races.sqlite`（リポジトリルート基準）。
- **実在確認**: 起動時に `_resolve_db_path()` で確認。本実行では既存予測を利用。

## 評価ロジック

- 選抜: d_hi475（skip_top20pct + 4.30≤EV<4.75 + prob≥0.05）。
- 運用: switch_dd4000。比較: normal_only。
- n_windows=36。月別は test_start の年月で集計。block_size=6。

## 実装内容

- ツール: `kyotei_predictor/tools/run_exp0052_seasonality_analysis.py`
- 月別・ブロック別・窓別累積曲線・switch 発動・regime（勝ちブロック vs 負けブロックの bet_count / hit_rate / 平均 EV）を算出し JSON とコンソールで出力。

## 実行コマンド

```bash
python3 -m kyotei_predictor.tools.run_exp0052_seasonality_analysis --n-windows 36
```

## 8. Regime 解釈

- **勝ちブロック（2ブロック）**: 平均 bet_count 189、hit_rate 7.36%、平均 EV 4.5316。
- **負けブロック（4ブロック）**: 平均 bet_count 163.2、hit_rate 8.34%、平均 EV 4.5268。
- **負け期間の共通点**:
  - **bet_count**: 負けブロックの方が少ない（163 vs 189）。弱い期間は選抜に乗るレース数がやや少ない可能性。
  - **hit_rate**: 負けブロックの方が高い（8.34% vs 7.36%）。当たり数では悪くないため、**払い戻し（オッズ）が低い当たりに偏っている**可能性がある。
  - **EV 帯**: 平均 EV はほぼ同水準（4.53 前後）。EV 帯のシフトでは説明しきれない。
- **結論**: 弱い期間は「bet 数がやや少ない」「hit_rate はむしろ高いが、利益が出ない」という特徴。オッズ分布や的中時のペイアウトの regime 差が疑われる。EV 帯の平均は安定。
