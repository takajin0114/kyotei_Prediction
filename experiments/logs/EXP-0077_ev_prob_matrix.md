# EXP-0077: EV band × probability band 最適領域探索

## 実験目的

- 現行ベースライン（4.30 ≤ EV < 4.75, 0.05 ≤ prob < 0.09）が本当に最良帯かを確認する。
- EV帯と確率帯の組み合わせごとに、**どの領域に実運用上の歪み（利益源・損失源）があるか**を定量化する。
- ROI だけでなく total_profit / max_drawdown / profit_per_1000_bets / bet_count のバランスが最もよい領域を見つける。

## 実装内容

- **ツール**: `kyotei_predictor/tools/run_exp0077_ev_prob_matrix.py`
- **出力**: `outputs/ev_prob_matrix/exp0077_ev_prob_matrix.json`（outputs は gitignore のため**未コミット**）
- 既存 calib_sigmoid 予測を利用し、EV band × prob band の 36 条件（baseline 含む）を n_windows=36 で評価。

## 共通条件

- calibration: sigmoid
- risk_control: switch_dd4000
- skip_top20pct: true
- **ref_profit**: EXP-0070 と同一（4.30 ≤ EV < 4.75, prob ≥ 0.05）で算出した schedule を全 CASE 共通で使用

## 比較設計

- **baseline**: 4.30 ≤ EV < 4.75, 0.05 ≤ prob < 0.09（CASE0_baseline）
- **EV band 候補**: (4.10,4.50), (4.20,4.60), (4.30,4.75), (4.30,4.60), (4.40,4.80), (4.50,4.90)
- **prob band 候補**: (0.05,0.07), (0.05,0.08), (0.05,0.09), (0.05,0.10), (0.06,0.09), (0.07,0.09)
- 上記の組み合わせのうち重複を除き **36 CASE**（CASE0_baseline + 35 ユニークな EV×prob 帯）を評価。

## 採用判断

- **総合で baseline を上回る領域**: **4.50 ≤ EV < 4.90**（ev450_490）かつ **0.05 ≤ prob < 0.09** が、total_profit・ROI・profit_per_1000_bets のバランスで最良。
  - **CASE_ev450_490_prob005_009**: total_profit 17,338・ROI 159.36%・max_dd 5,400・bet_count 124。**実運用候補として adopt**。現行 baseline（4.30–4.75, 0.05–0.09）より EV 上限を 4.90 に広げることで利益・効率が改善。
  - **CASE_ev450_490_prob005_008**: profit 13,558・bet 103。**hold**（攻め用オプション）。
  - **CASE_ev450_490_prob007_009**: ROI 189.57% だが bet_count 43 のみ。ROI 単独では判断せず **hold**（サンプル数少のため過学習リスクを注記）。
- **baseline（4.30–4.75, 0.05–0.09）**: profit 8,870・bet 160。ev450_490_prob005_009 に劣るが、EV 帯を広く取りたい場合は現行維持で **hold**。
- **EV 帯 4.30–4.60（ev430_460）**: 全 6 条件で -100%。**reject**。実運用ではこの狭い EV 帯は避ける。
- **低 EV 帯（4.10–4.50, 4.20–4.60）**: prob を絞ると（0.07–0.09 等）黒字になるが、total_profit は ev450_490 より小さい。**hold**（サブ候補）。

## 考察

- **市場の歪み**: 高 EV 帯（4.50–4.90）かつ中〜低確率（0.05–0.09）に利益が集中。逆に EV 上限を 4.60 に絞ると全 prob で大損失となり、**EV 帯の取り方**が効くことが定量化された。
- **0.05 ≤ prob < 0.09 の妥当性**: ベースラインと同帯（prob 0.05–0.09）を EV だけ 4.50–4.90 に広げた ev450_490_prob005_009 が total_profit 最大。prob 帯「0.05–0.09」は維持し、**EV 上限を 4.75 → 4.90 に拡張**する選択肢が有効。
- **次の考察**: n_windows=24/30 での頑健性確認、および ev450_490 を新主軸候補とした場合の ref_profit との整合性確認。
