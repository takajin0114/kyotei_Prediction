# EXP-0073: EV Distribution Analysis

## 実験目的

主軸戦略（4.50≤EV<4.75, prob≥0.05）の ROI 11.12% は確認済みだが、**EV帯ごとの利益分布**は未確認であった。EV distribution × ROI を分析し**利益源EV帯**を特定し、次の戦略（EXP-0074 以降）の方向を決める。

## EV帯定義

baseline 分析範囲: **4.30 ≤ EV < 6.00**, prob ≥ 0.05。EV を以下 12 帯で分割（左閉右開）。

|----|------|

## 実装内容

- **ツール**: `kyotei_predictor/tools/run_exp0073_ev_distribution_analysis.py`
- **出力**: `outputs/ev_distribution/exp0073_ev_distribution.json`, `exp0073_ev_distribution.csv`（outputs は gitignore のため未コミット）
- 共通条件: calibration=sigmoid, risk_control=switch_dd4000, n_windows=36, skip_top20pct。ref_profit は 4.30≤EV<4.75 で算出（EXP-0070 と同一）。
- baseline: 4.30 ≤ EV < 6.00, prob ≥ 0.05, difficulty filter なし。各 bet を EV 帯に振り分け、帯ごとに ROI / total_profit / bet_count / hit_rate / avg_odds / profit_per_1000_bets を集計。

## 利益源EV帯

- **黒字は 4.50-4.75 のみ**。ROI 11.12%、total_profit 5,772、bet_count 590、profit_per_1000_bets 9,783.05。
- 4.30-4.50: 赤字（ROI -13.5%、profit -5,288）。主軸に含めない方がよい。
- 4.75-5.00: 小幅赤字（-1,046）。拡張は慎重に。
- 5.00-6.00: 大幅赤字（-49,294、bet 1,903）。高EV帯の過大評価が強く、現状モデルではこの帯は避けるべき。

## 考察

- 主軸戦略（4.50≤EV<4.75）は**唯一の利益源EV帯**に絞っており、分析結果と一致する。
- baseline を 4.30–6.00 に広げると、4.30-4.50 / 4.75-5.00 / 5.00-6.00 の損失が加わり、全体では大幅赤字になる。EV 上限を 4.75 に固定する現行設計は妥当。
- 次の戦略（EXP-0074 以降）では、4.50-4.75 の維持または微調整（prob 閾値・難易度フィルタの再検討）が中心となり、EV 帯の拡大（特に 4.75 以上・4.30–4.50 の本格採用）は推奨しない。
