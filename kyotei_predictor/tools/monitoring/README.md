# Monitoring Tools

競艇予測システムの監視・モニタリングツール群です。システム性能・予測精度・最適化状況の監視を担当します。

## 📁 ファイル構成

### 主要監視ツール
- `hit_rate_monitor.py` - 的中率監視・分析ツール
- `performance_monitor.py` - システム性能監視ツール
- `monitor_optimization.py` - 最適化プロセスの監視
- `check_optimization_status.py` - 最適化状況の確認
- `simple_monitor.py` - シンプルな監視ツール

## 🚀 使用方法

### 的中率監視
```bash
# 的中率の監視・分析
python tools/monitoring/hit_rate_monitor.py

# 特定期間の的中率分析
python tools/monitoring/hit_rate_monitor.py --start-date 2024-01-01 --end-date 2024-01-31
```

### 性能監視
```bash
# システム性能の監視
python tools/monitoring/performance_monitor.py

# リアルタイム監視
python tools/monitoring/performance_monitor.py --real-time
```

### 最適化監視
```bash
# 最適化プロセスの監視
python tools/monitoring/monitor_optimization.py

# 最適化状況の確認
python tools/monitoring/check_optimization_status.py
```

## 📊 監視機能

### 的中率監視
- **予測精度**: 3連単予測の的中率・精度分析
- **時系列分析**: 的中率の推移・トレンド分析
- **会場別分析**: 会場別の的中率・精度比較
- **条件別分析**: 天候・機材等の条件別分析
- **アラート機能**: 的中率低下時の自動通知

### システム性能監視
- **処理速度**: データ取得・予測・分析の処理時間
- **リソース使用量**: CPU・メモリ・ディスク使用量
- **スループット**: 単位時間あたりの処理件数
- **エラー率**: 処理エラー・失敗率の監視
- **可用性**: システム稼働率・ダウンタイム

### 最適化監視
- **最適化進捗**: 試行回数・完了率・残り時間
- **スコア推移**: 最適化スコアの改善状況
- **パラメータ変化**: ハイパーパラメータの最適化状況
- **収束性**: 最適化の収束・安定性評価
- **異常検知**: 最適化プロセスの異常・停滞検知

## 🔧 技術仕様

### 監視データ
- **ログファイル**: 構造化ログ・エラーログ
- **メトリクス**: 性能指標・統計データ
- **アラート**: 閾値超過・異常検知
- **レポート**: 定期レポート・サマリー

### 監視間隔
- **リアルタイム**: 1秒～1分間隔の即座監視
- **短周期**: 5分～1時間間隔の定期監視
- **長周期**: 日次・週次・月次の定期監視
- **イベント駆動**: 特定イベント発生時の監視

### アラート機能
- **閾値設定**: カスタマイズ可能なアラート閾値
- **通知方法**: ログ・メール・Slack等の通知
- **エスカレーション**: 重要度に応じた通知レベル
- **抑制機能**: 重複アラート・スパム防止

## 📈 監視指標

### 的中率指標
- **全体的中率**: 全予測の的中率
- **会場別的中率**: 会場ごとの的中率
- **条件別的中率**: 天候・機材等の条件別的中率
- **的中率推移**: 時系列での的中率変化
- **的中率分散**: 的中率のばらつき・安定性

### 性能指標
- **処理時間**: 各処理の実行時間
- **スループット**: 単位時間あたりの処理件数
- **リソース効率**: CPU・メモリ使用効率
- **スケーラビリティ**: 負荷増加時の性能変化
- **安定性**: 長時間運用時の性能安定性

### 品質指標
- **データ品質**: データの完全性・正確性
- **モデル品質**: 予測モデルの性能・安定性
- **システム品質**: システムの信頼性・可用性
- **運用品質**: 運用プロセスの効率性・品質
