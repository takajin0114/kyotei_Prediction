## 運用ルール
- 共通処理・定数・設定はcommon/に集約
- 他ツールからimportして利用
- 不要な一時ファイルは随時削除

### 新規共通処理の追加・変更時の運用ルール
- 新たな共通処理（定数、汎用関数、API共通処理など）は必ずcommon/に追加し、他ディレクトリに重複実装しない
- 共通処理の追加・変更時はREADMEの「ファイル構成」「使い方」セクションも必ず更新
- 共通処理のテストはtest_env.pyやtests/配下で実施
- 他ツールからimportする際は、パスや依存関係に注意
- 不要になった共通処理は随時削除し、READMEからも除外

# Common Utilities

共通ユーティリティ・ヘルパー関数群です。全ツールで共有される機能を提供します。

## 📁 ファイル構成

- `venue_mapping.py` - 会場コード・名前マッピング
- `config.py` - 設定管理
- `logger.py` - ログ機能
- `utils.py` - 汎用ユーティリティ
- `test_env.py` - Python環境・依存パッケージ・パス設定などの動作確認用スクリプト

## 🚀 使用方法

### 会場マッピング
```python
from tools.common.venue_mapping import get_venue_name, get_venue_code
venue_name = get_venue_name("KIRYU")
venue_code = get_venue_code("桐生")
```

### 設定管理
```python
from tools.common.config import Config
config = Config()
data_dir = config.get_data_dir()
```
