---

## 🎯 基本方針

### 最終目標
**3連単の各組み合わせ（120通り）に対して確率で予測し、オッズと比較して投資対象を決定する**

### 現在の状況
- **1位的中率**: 2-4%（理論値0.83%の2-5倍）
- **上位5位的中率**: 16-18%
- **上位10位的中率**: 26-28%
- **最良アルゴリズム**: equipment_focused

## 🚀 精度向上戦略

### Phase 1: データ拡張・品質向上 ⭐ **最優先**

#### 1.1 データ量の拡大
- **目標**: 10,000レース以上のデータ収集
- **期間**: 3-6ヶ月
- **効果**: 統計的信頼性の向上、パターン発見の可能性

#### 1.2 データ品質の向上
- **着順情報の完全性**: 99.9% → 100%
- **機材データの詳細化**: 交換履歴、直近成績
- **環境データの追加**: 天候、風向、波高の詳細

#### 1.3 特徴量エンジニアリング
```python
# 新規特徴量の追加
- 選手の直近調子（過去10レースの成績）
- 機材の相性スコア（選手×機材の組み合わせ成績）
- コース特性スコア（内側・外側コースの適性）
- 時間帯効果（午前・午後の成績差）
- レース間隔効果（前レースからの時間）
```

### Phase 2: アルゴリズム高度化

#### 2.1 機械学習モデルの導入
- **モデル候補**: 
  - Random Forest（特徴量の重要度分析）
  - XGBoost（勾配ブースティング）
  - Neural Network（深層学習）
- **特徴**: 非線形関係の学習、特徴量の自動選択

#### 2.2 アンサンブル手法
- **複数アルゴリズムの組み合わせ**
- **重み付き平均による予測精度向上**
- **バギング・ブースティングの活用**

#### 2.3 時系列分析の導入
- **選手の調子の変化パターン**
- **機材性能の時系列変化**
- **季節性・周期性の考慮**

### Phase 3: 3連単特化モデル

#### 3.1 着順依存性の考慮
```python
# 現在: 独立確率の積
P(1-2-3) = P(1着) × P(2着) × P(3着)

# 改善: 条件付き確率
P(1-2-3) = P(1着) × P(2着|1着) × P(3着|1着,2着)
```

#### 3.2 艇間相関のモデル化
- **特定の艇同士の着順パターン**
- **コース位置による相互影響**
- **選手間の相性・ライバル関係**

#### 3.3 ベイズ推定の活用
- **事前確率と事後確率の更新**
- **不確実性の定量化**
- **信頼区間の計算**

## 💰 投資戦略の確立

### 期待値理論に基づく投資判断

#### 基本公式
```
期待値 = 的中確率 × オッズ
投資価値 = 期待値 > 1.0
```

#### 投資戦略の階層化

**Level 1: 保守的戦略**
- 期待値 > 1.5 の組み合わせのみ投資
- 上位5位的中率18%を活用
- リスク管理重視

**Level 2: バランス戦略**
- 期待値 > 1.2 の組み合わせを投資
- 上位10位的中率26%を活用
- リスク・リターンのバランス

**Level 3: 積極的戦略**
- 期待値 > 1.0 の組み合わせを投資
- 高配当狙いの組み合わせも含む
- 高リスク・高リターン

### ポートフォリオ戦略

#### 分散投資
- **複数組み合わせへの投資**
- **期待値の高い組み合わせを優先**
- **リスク分散による安定性向上**

#### 資金管理
- **総投資額の制限（例: 1レースあたり最大10,000円）**
- **組み合わせごとの投資額配分**
- **期待値に応じた投資額調整**
