# 詳細結果ファイル

このディレクトリには、最適化の詳細評価結果が保存されます。

## ファイル形式

- `graduated_reward_optimization_YYYYMM_YYYYMMDD_HHMMSS_detailed.json`
  - 最良モデルの詳細評価結果
  - 的中率、報酬分布、エピソード長などの詳細情報

## 自動コミット

このディレクトリ内のファイルは自動的にGitのコミット対象となります。
重要な結果ファイルとして管理されます。

## ファイル内容

各詳細結果ファイルには以下の情報が含まれます：

- **的中率**: 全体、的中タイプ別
- **報酬統計**: 平均、標準偏差、分布
- **エピソード長**: 平均エピソード長
- **的中タイプ別分析**: 完全的中、2着的中、1着的中、不的中
- **ベット統計**: 総ベット数、的中数

## 使用方法

```bash
# 結果ファイルの確認
ls results/

# 最新の結果ファイルを確認
ls -t results/ | head -1
```
