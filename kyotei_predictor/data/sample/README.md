# sample ディレクトリ

このディレクトリは、外部から取得したレースデータ・オッズデータのサンプル（開発・テスト用）を格納します。

## 用途
- 開発・テスト・動作確認用のサンプルデータ
- 大量アクセス防止のため、必要最小限のデータのみを格納
- 本番運用や大規模学習には raw/ ディレクトリ等の本データを利用

## サンプル内容
- 2025-07-04_KIRYU の全12R分のレースデータ・オッズデータ
- ファイル例: `race_data_2025-07-04_KIRYU_R1.json`, `odds_data_2025-07-04_KIRYU_R1.json` など

## 注意事項
- サンプルデータは著作権・利用規約に注意し、再配布・商用利用は避けてください
- 大量アクセスによるデータ取得は控えてください
- 必要に応じてサンプルデータを追加・差し替え可能

## 備考
- 本ディレクトリ配下は .gitignore で除外されていないため、git管理対象です
- 大容量データは raw/ や backup/ で管理し、sample/ には最小限のみ格納
