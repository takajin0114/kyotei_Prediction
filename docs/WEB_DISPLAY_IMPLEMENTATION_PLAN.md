**最終更新日**: 2025-07-15

## 🎯 概要

予測結果（JSON形式）をWebブラウザで見やすく表示する機能を実装します。  
現在の予測結果は `outputs/predictions_YYYY-MM-DD.json` 形式で保存されており、これを静的HTMLファイルで表示します。

### 対象データ構造
- **予測日**: `prediction_date`
- **会場別予測**: `predictions[]` - 各レースの3連単上位20組
- **購入提案**: `purchase_suggestions[]` - 流し・ボックス・単勝買い
- **会場サマリー**: `venue_summaries[]` - 会場別統計

- **probability**: 確率（0-1の値）
- **expected_value**: 期待値（プラス/マイナス）
- **rank**: 順位（1-20）

#### 購入提案
- **type**: 提案タイプ（"wheel", "box", "nagashi"）
- **description**: 提案説明
- **combinations**: 組み合わせリスト
- **total_probability**: 合計確率

- **expected_return**: 期待リターン
