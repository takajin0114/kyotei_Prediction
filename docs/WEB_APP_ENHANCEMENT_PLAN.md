## 📊 実装詳細

### 1.1 履歴タブ実装

#### データ構造
```json
{
  "prediction_history": [
    {
      "id": "2025-07-06_KIRYU_R1",
      "date": "2025-07-06",
      "stadium": "KIRYU",
      "race_number": "R1",
      "algorithm": "equipment_focused",
      "predictions": [...],
      "actual_result": "2-1-4",
      "accuracy": {
        "hit_rank": 1,
        "hit_probability": 17.50,
        "algorithm_performance": "excellent"
      }
    }
  ]
}
```

#### 表示項目
- **基本情報**: 日付、場、レース番号
- **予測結果**: 上位3位の予測
- **実際結果**: 実際の着順
- **的中情報**: 順位、確率、性能評価
- **アルゴリズム**: 使用したアルゴリズム

#### フィルタリング機能
- 日付範囲指定
- 場別フィルタ
- アルゴリズム別フィルタ
- 的中率別フィルタ

### 1.2 分析タブ実装

#### 機材詳細分析
- **基本機材スコア**: ボート・モーター2連率
- **相性スコア**: ボート-モーター相性
- **トレンド分析**: 時系列性能変化
- **組み合わせ効果**: 相乗効果の可視化
- **選手適応性**: 機材適応能力

#### 可視化要素
- **棒グラフ**: 各艇の機材スコア比較
- **ヒートマップ**: 相性マトリックス
- **折れ線グラフ**: 時系列トレンド
- **円グラフ**: 重み配分の表示
- **テーブル**: 詳細数値データ

#### インタラクティブ機能
- 艇番号クリックで詳細表示
- スコア項目の詳細説明
- 比較モード（複数艇同時表示）
- エクスポート機能

### 1.3 データ保存機能

#### 保存形式
```python
# 予測結果の保存
def save_prediction_result(race_data, prediction_result):
    history_entry = {
        'timestamp': datetime.now().isoformat(),
        'race_info': race_data['race_info'],
        'prediction': prediction_result,
        'algorithm': prediction_result['algorithm'],
        'execution_time': prediction_result['execution_time']
    }
    
    # JSONファイルに保存
    save_to_json(history_entry)
```

#### ファイル構造
```
kyotei_predictor/
├── data/
│   ├── history/
│   │   ├── predictions_2025-07-06.json
│   │   └── analysis_cache.json
│   └── ...
```
