# 月次レポート（3.4.2）

月ごとに「学習設定・使用データ・検証結果」を 1 ファイルにまとめて残すためのフォルダです。手動で記入しても構いません。

## 使い方

1. `TEMPLATE.md` をコピーし、ファイル名を `YYYY-MM.md`（例: `2026-03.md`）にリネームする。
2. 各セクションをその月の内容で埋める。
3. 必要に応じて `logs/` のログや `outputs/` の予測結果を参照する。

## ファイル名

- **TEMPLATE.md** … 記入用テンプレート（コピーして使う）
- **YYYY-MM.md** … 各月のレポート（例: 2026-03.md）
