# 競艇予測Webアプリケーション - 要件定義書

## 📋 プロジェクト概要

### 目的
競艇（ボートレース）の予測を支援するWebアプリケーションの実装

### 対象ユーザー
- 競艇ファン
- 予想を楽しむ一般ユーザー
- データ分析に興味のあるユーザー

## 🎯 機能要件

### 1. 基本機能（MVP: Minimum Viable Product）

#### 1.1 メインページ表示
- **目的**: アプリケーションの基本動作確認
- **内容**: 
  - タイトル表示
  - ナビゲーション
  - 基本レイアウト
- **優先度**: 🔥 P0（最優先）

#### 1.2 レースデータ表示
- **目的**: 既存データの可視化
- **内容**:
  - 出走表の表示
  - 選手情報（名前、級別、勝率）
  - ボート・モーター情報
  - 天候情報
- **データソース**: 既存の`complete_race_data_20240615_KIRYU_R1.json`
- **優先度**: 🔥 P0（最優先）

#### 1.3 基本予測機能
- **目的**: シンプルな予測アルゴリズムの実装
- **内容**:
  - 勝率ベースの順位予測
  - 予測結果の表示
- **アルゴリズム**: 
  ```
  予測スコア = 全国勝率 × 0.6 + 当地勝率 × 0.4
  ```
- **優先度**: ⚡ P1（高優先）

### 2. 拡張機能

#### 2.1 高度な予測機能
- **目的**: より精度の高い予測
- **内容**:
  - 複数要素を考慮した予測
  - 天候補正
  - 機材成績の考慮
- **アルゴリズム**:
  ```
  予測スコア = 全国勝率×0.4 + 当地勝率×0.3 + 機材成績×0.2 + 天候補正×0.1
  ```
- **優先度**: ⚡ P1（高優先）

#### 2.2 予想保存・履歴管理
- **目的**: ユーザーの予想記録
- **内容**:
  - 予想結果の保存
  - メモ機能
  - 履歴一覧表示
- **データ形式**: JSON形式（`predictions.json`）
- **優先度**: 📈 P2（中優先）

#### 2.3 統計分析・グラフ表示
- **目的**: データの視覚化
- **内容**:
  - 勝率の棒グラフ
  - スタートタイムの折れ線グラフ
  - 選手比較チャート
- **ライブラリ**: Chart.js
- **優先度**: 📈 P2（中優先）
